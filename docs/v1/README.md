# 相册整理 App — v1 规格总览

> **基线版本**：v1 核心整理流已在 v1.1 中延续并扩展。当前产品名 **相册主理人**，功能索引见 [../v1.1/README.md](../v1.1/README.md)。

> Flutter · iOS only · Tinder 式滑动整理系统相册

## 一句话

帮助用户快速整理 iPhone 系统相册：选来源 → 逐张滑动 → 右滑归类、左滑标记删除 → 统一确认删除。

## 核心决策摘要

| 维度 | v1 决策 |
|------|---------|
| 平台 | 仅 iOS |
| 相册 | 读写系统 Photo Library |
| 整理流程 | 先选来源相册 + 默认目标相册，再滑动 |
| 右滑 | 加入目标系统相册，保留在来源，本地标记已处理 |
| 左滑 | 进入待删除队列，不立即删除 |
| 已处理照片 | 本地 DB 记录，默认不再出现，可「重新整理」 |
| 撤销 | 最近 1 次滑动，Snackbar + 撤销按钮 |
| 权限 | 兼容完全访问 + 有限访问 |
| 语言/主题 | 简体中文，跟随系统深浅色 |
