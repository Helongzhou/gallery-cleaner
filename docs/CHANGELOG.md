# Changelog

本文件记录 **相册主理人** 面向用户与开发者的版本变更。规格细节见 `v1.1/` 各模块文档。

## [1.1.0] — 2026-06

### 新增

- **四 Tab 导航**：相库、智能、足迹、我的
- **滑动撤销增强**：当前会话内最多撤销 3 步
- **待删除页**：瀑布流、体积摘要、移出待删 / 彻底删除双 CTA
- **智能 · 截图清理**：30 / 90 / 365 天档位，Isolate 扫描 + SQLite 缓存
- **足迹 Tab**：`flutter_map` 地图打点、城市列表、照片半屏、分享海报（含地图截图）
- **整理历史**：按会话时间线展示，抽屉内二级查看照片与逐条反悔
- **我的**：主题、足迹地图样式（含跟随系统）、缓存清理、关于与反馈
- **AppBar 操作**：相库历史、智能雷达强制刷新
- **UniversalModal**：统一确认弹窗；`AppText.formatMixed` 文案格式化
- **品牌**：应用显示名「相册主理人」
- **应用图标**与**浅/深启动图**
- **Android 支持**（API 29+）：全功能对标 iOS v1.1，侧载 APK 分发

### 变更

- 「共享」Tab 改为「足迹」；`/shared` 重定向至 `/footprints`
- 设置从独立页合并至「我的」Tab，移除 `/settings` 路由
- 主题与足迹地图样式均支持「跟随系统」
- `PhotoLibraryService` 增加 Android 平台分支（创建相册、撤销移出、截图相册识别）

### 未做（规划后续）

- 相似照片、超大视频、模糊检测
- Face ID 真实验证（`local_auth`）、隐藏相册

## [1.0.0] — 基线

### 核心能力

- 来源 / 目标相册选择，Tinder 式滑动整理
- 右滑归入目标相册，左滑标记待删除
- 待删除队列与系统相册删除确认
- 撤销最近 1 次操作、断点续整理
-  onboarding、权限与有限访问提示
