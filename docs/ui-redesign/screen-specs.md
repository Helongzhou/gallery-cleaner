# 逐屏规格

## 1. 引导页（Onboarding）

**结构不变，视觉升级**

- 背景 `background`
- 每步：大图标区（圆角 24 容器）+ `largeTitle` 标题 + `body` 说明
- 底部：`跳过` 文字按钮（左）+ `开始使用` 主按钮（右，圆角 16）
- 第一步嵌入来源/目标选择示意图（静态 asset 或代码绘制 mock）

---

## 2. 首页（相库 Tab）

### 顶部
- Large Title「整理相册」左对齐，34pt Bold
- 右上角圆形头像按钮（40×40，`surfaceContainer` 背景）→ v1 占位 Toast

### 来源相册区
- Section 标题：`headline`「来源相册」
- **16:10 大卡片**：
  - 封面：来源相册最新一张照片
  - 底部渐变遮罩 + 白色文字：相册名 + 总张数
  - 橙色徽章「待整理: N」
  - 装饰性双层旋转阴影（`primary/5` + `primary/10`）
  - 点击 → 底部 Sheet 相册列表
- Sheet 每项：相册名 + 总张数 + 待整理数，选中打勾

### 目标相册区
- Section 标题 + 右侧「查看全部」链接（`primary`）
- **横向滚动**：
  - 首项：虚线边框「新建相册」（112×112）
  - 其余：方形缩略图（112×112）+ 底部相册名
  - 选中：3px `#0058BC` ring
- 「查看全部」→ 底部 Sheet 全部相册

### 待删除入口
- `surfaceContainerHigh/50` 圆角卡片
- 左侧红色圆形图标 + 标题「待删除照片」+ 副标题「N 张待确认」
- 右侧 chevron

### 底部固定区
- 毛玻璃底栏（`BackdropFilter` blur 20）
- 全宽主按钮「开始整理」（高 56，圆角 16，带 ✨ 图标）
- 待整理为 0 时按钮禁用，显示「已全部整理」
- Tab Bar 4 项：相库(激活) / 智能 / 共享 / 我的

### 其他
- 有限访问：顶部 `MaterialBanner` 样式对齐 token
- 断点续整理：来源卡片上方提示条「继续整理？还剩 N 张」

---

## 3. 滑动整理页（全屏，隐藏 Tab Bar）

### 顶部导航（毛玻璃）
- 左：`‹ 返回`（primary 色）
- 中：来源相册名 + `footnote` 进度「12 / 450」
- 右：`more_horiz` 占位
- 下方 3px 进度条（`primary` 填充，`surfaceVariant` 轨道）

### 卡片区
- **3:4 比例**，圆角 24，阴影 `shadow-xl`
- 背景堆叠卡片（90% 宽度，scale 0.95，opacity 0.5）
- 底部渐变遮罩：拍摄日期（`footnote`，白色）
- v1 不显示地点

### 滑动反馈
- 右滑：绿色印章「移入相册」（边框 4px `systemGreen`，旋转 15°）
- 左滑：红色印章「标记删除」（旋转 -15°）
- 透明度随滑动距离 0→1

### 底部
- 左/右滑图标提示（半透明，居中）
- 右下：撤销胶囊按钮（`surfaceContainerHighest` 背景，圆角 full）
- v1 无分享/收藏按钮

---

## 4. 整理完成页（全屏）

### 头部
- 绿色勾选圆圈动画（`systemGreen/10` 背景，bounce 0.6s）
- `largeTitle`「整理完成!」
- `subheadline` 副标题

### Bento 网格（2 列）
- **已归类卡片**：数字（primary 色）+ 目标相册最新 2 张缩略图堆叠 +「+N」
- **待删除卡片**：数字（systemRed）+ 红色装饰进度条

### 操作区
- 主按钮「去清理待删除照片」（有待删除时显示）
- 次要「整理其他相册」
- 文字链「返回首页」

---

## 5. 待删除确认页（全屏）

### 导航栏
- 左「取消」| 中「待删除确认」| 右「全选」/「取消全选」

### 安全提示
- `systemGray6/50` 背景条
- info 图标 +「确认后，照片将从所有设备中移除」

### 照片网格
- 3 列，间距 2px（无缝拼图）
- 选中：右下角红色圆形 ✕ 徽章 + 红色半透明描边
- 默认全选

### 底部固定
- 毛玻璃底栏
- 红色全宽按钮「确认删除 N 张照片」
- 说明「照片将移入系统最近删除相册」
- 恢复：导航栏「恢复」或在选中时显示次要操作

---

## 6. 占位 Tab 页

- 智能 / 共享 / 我的：居中图标 +「即将推出」
- 点击 Tab 时 `HapticFeedback.selectionClick()`
